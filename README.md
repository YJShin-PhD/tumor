# Tumor Growth Analysis System
이 프로젝트는 종양 성장 데이터를 자동으로 전처리하고, 통계 분석 및 그래프를 생성하는 시스템입니다.

## 주요 기능
* **데이터 자동 전처리**: 가로형 엑셀 데이터를 분석용 세로형 데이터로 자동 변환.
* **통계 분석**: 마지막 측정일 기준 G1 vs G2 그룹 간의 T-test (p-value 계산).
* **성장률 계산**: 각 개체별 일일 성장률(%) 산출.
* **시각화**: 표준 오차(SE) 막대가 포함된 종양 성장 곡선 그래프 생성.

## 사용 방법
1. `result.xlsx`와 같은 데이터 파일을 폴더에 넣습니다.
2. `python master_process.py`를 실행합니다.
